==>Git status: cria repositorio local

==>Git config --global user.name "fulano-de-tal": configura caonta de usuário no git
==>Git config --global user.email "fulano@teste.com": configura email do usuário git
==> git reset  [arquivo.ext]: remove arquivo da area staging
==> git add  [arquivo.ext]: adiciona arquivo da area staging
